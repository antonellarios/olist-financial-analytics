# Financial Analytics Case Study – Olist E-commerce

This project analyzes the financial performance of a Brazilian e-commerce platform
using transactional data from Olist.

The objective is to demonstrate how **data analytics and accounting principles**
can be combined to generate reliable financial insights and simulate reporting processes.

---

## Project Scope

- Revenue recognition based on delivered orders
- Data quality audit
- Monthly revenue analysis (time-adjusted)
- Double-entry accounting simulation
- Trial balance validation
- Profitability analysis (logistics contribution margin)
- Waterfall chart for revenue erosion analysis

---

## Key Concepts Applied

- Financial data validation
- Accounting logic (double-entry, trial balance)
- Business-oriented storytelling
- Data visualization for financial decision-making

---

- Dataset: Olist Brazilian E-commerce
- Source: Kaggle
